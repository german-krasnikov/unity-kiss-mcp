## [v0.8.0] â€” 2026-06-04 <!-- svg: compile auto-fix + editor-state injection + tool ping -->

- **Compile Auto-Fix Retry Loop (Plugin 0.8.0, #5)** (2026-06-04) â€” `CompileAutoFix.cs` watches `EditorApplication.CompilationFinished` events and auto-retries up to 3 times when chat edits compile. On domain-reload resume, the snapshot is prepended to sent text via `SentTextCache`, eliminating the 2â€“3 cold-start probe calls Claude used to make. - **Tool Ping on Call Complete (Plugin 0.8.0, #29)** (2026-06-04) â€” `ToolPing.cs` flashes any GameObject a tool call touches via `EditorGUIUtility.PingObject`.
